# easyIT Nachhilfe Leipzig – Phase 1

## Enthalten
- PHP-Grundgerüst mit Includes
- getrennte responsive CSS-Dateien für Header, Sidebar, Inhalt und Footer
- Leipzig-Blau/Gold-Neon-Design
- mobile Navigation
- nojquery-Kurzzeichen `_`
- Speicherung des letzten Navigationslinks
- SEO-Meta-Daten je Seite
- Canonical URLs
- Open Graph und Twitter Card
- JSON-LD für EducationalOrganization / LocalBusiness
- robots.txt und sitemap.xml
- Basis-Seiten für die vier MINT-Fächer, Methodik, FAQ, Jobs, Kontakt und Rechtliches
- .htaccess für Kompression, Caching und Sicherheitsheader

## Vor Veröffentlichung ändern
1. Domain in `config/site.php`
2. Telefonnummer, E-Mail und vollständige Adresse
3. Impressum und Datenschutz rechtlich vervollständigen
4. Sitemap-URLs prüfen
5. PHP 8.1 oder neuer verwenden

## Lokaler Start
Im Projektordner:
`php -S localhost:8080`

Dann:
`http://localhost:8080`
