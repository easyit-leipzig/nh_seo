# Installation Phase 5 unter /nh_seo/

Diese Phase ergänzt ausschließlich neue Inhalte in den Verzeichnissen:

- `blog/`
- `downloads/`
- `tools/`

Es werden keine bestehenden PHP-, CSS-, JavaScript-, Include- oder Konfigurationsdateien aus den Phasen 1–4 überschrieben.

## Installation

Den Inhalt dieses ZIP-Archivs direkt nach folgendem Verzeichnis entpacken:

```text
htdocs/nh_seo/
```

Danach liegen die neuen Inhalte beispielsweise unter:

```text
htdocs/nh_seo/blog/
htdocs/nh_seo/downloads/
htdocs/nh_seo/tools/
```

Die Basis-URL der lokalen Website bleibt:

```text
https://localhost/nh_seo/
```

Hinweis: Die Dateien dieser Phase sind redaktionelle Markdown-Vorlagen und noch keine direkt gerenderten PHP-Webseiten.
